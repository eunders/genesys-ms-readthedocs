2. Dashboard
============

.. image:: /_static/en/dashboard.jpg
    :alt: Optional alt text for the image
    :align: center
    :width: 500px

The Dashboard provides an overview of key metrics and system status.

2.1 Dashboard Overview
----------------------

The dashboard displays important metrics, including:

- Number of Users
- Orders
- Amount
- Visitors
- Leads
- Conversions

2.2 Menu Bar
------------

The menu bar on the left-hand side provides access to the following options:

- **User**
- **Site**
- **CRM**
- **WhatsApp**

These options offer various features, which will be discussed in the following sections.
